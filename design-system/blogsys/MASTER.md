# BlogSys 设计系统

> 本文件是前端主题的长期约束。页面级规则仅在明确记录时覆盖这里。

## 产品与用户

- 产品：面向多用户的阅读、写作与交流平台。
- 用户：各年龄段普通读者、作者和管理员。
- 气质：克制、干净、有编辑感，优先保证阅读与操作清晰。

## 视觉方向

采用柔和的 Swiss 编辑设计：白色和中性灰表面、Helvetica 系无衬线字体、单一 Yves Klein Blue 强调色、非侵入式细分隔线和充足留白。

核心表现是“杂志目录式信息层级”：标题、摘要、作者、日期和分类依靠排版层级组织，不使用巨大编号、满屏网格或卡片堆叠。

## 设计令牌

| 用途 | 值 |
|---|---|
| 页面背景 | `#F7F7F8` |
| 内容表面 | `#FFFFFF` |
| 主文字 | `#111111` |
| 次文字 | `#575757` |
| 弱文字 | `#737373` |
| 分隔线 | `#E2E2E5` |
| 强分隔线 | `#C8C8CC` |
| 唯一强调色 | `#002FA7` |
| 字体 | `Helvetica Neue, Helvetica, Arial, PingFang SC, Microsoft YaHei, sans-serif` |
| 内容最大宽度 | `1180px` |
| 正文阅读宽度 | `760px` |
| 交互过渡 | `180ms ease` |

间距序列：`4 / 8 / 12 / 16 / 24 / 32 / 48 / 72px`。

## 组件规则

- 按钮和输入只允许 `2–3px` 功能性圆角；内容容器保持直角。
- 主按钮使用蓝底白字，悬停变为主文字黑；禁止渐变按钮。
- 内容列表优先使用顶部分隔线，不为每条内容创建独立悬浮卡片。
- 表格表头使用浅灰底和深色文字，不使用整块黑色表头。
- 弹窗可使用轻微边框；仅下拉层允许非常克制的阴影。
- 图片保持自然颜色，不默认灰度化，不用缩放制造悬停效果。
- 蓝色只用于链接、焦点、选中状态和主要操作。

## 页面规则

- 首页首屏以标题、说明、真实统计和标准操作链接构成，不使用装饰性编号。
- 文章列表采用标题、摘要、元数据和缩略图的目录式排列。
- 文章正文宽度不超过 `760px`，正文约 `17–18px`、行高 `1.85–1.9`。
- 用户中心和后台使用轻量侧栏与开放工作区，不嵌套多层卡片。
- 登录注册表单保持单列、左对齐，字段使用标准文案。

## 禁止模式

- 大圆角、胶囊化布局和漂浮卡片墙。
- 渐变按钮、渐变背景、玻璃拟态和装饰性阴影。
- 黑色大色块、巨大编号、过粗边框和满屏网格。
- 没有业务含义的标签、虚构数据、主题化操作文案或 Emoji 图标。
- 依靠缩放造成布局跳动的悬停效果。

## 可用性门禁

- 正文和控件满足至少 4.5:1 对比度。
- 所有交互具备清晰 hover、focus-visible 和 pointer 状态。
- 可点击文章支持 Enter 与 Space 键。
- 图片必须提供与内容相关的 `alt`。
- 支持 `prefers-reduced-motion`。
- 验证宽度：375、768、1024、1440px；移动端不得横向滚动。
