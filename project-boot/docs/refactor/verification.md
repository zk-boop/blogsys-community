## 已验证的关键性质

- 匿名写接口返回 401，越权后台接口返回 403，登录用户写操作要求 CSRF。
- Session 不保存 JPA 用户实体；文章和评论作者不能由客户端 ID 冒充。
- 用户不能修改他人资料或文章，草稿不会公开给其他用户。
- 测试不访问共享或远程 MySQL，不依赖真实凭据。

## 告警与发布结论

- Vue 构建存在 vendor CSS/JS 超过推荐体积的性能告警，不阻断功能构建。
- 本轮代码可作为模块化重构基线；由于数据库版本迁移和真实生产拓扑验证尚未完成，不建议直接无演练发布到生产环境。
