# 07 文件模块

## 目标

- 上传必须登录，删除必须验证文件所有权或管理员权限。
- 允许列表校验 MIME、扩展名和文件大小。
- 规范化后的路径必须位于配置根目录内。
- 服务端生成文件名，下载使用安全响应头。

## 部署决策

单实例可保留本地卷；多实例部署应切换对象存储并记录迁移策略。

## 验证

覆盖匿名上传、伪造扩展名、路径穿越、超限文件、越权删除和缺失文件。

## 已实施

- 上传接口要求认证，删除接口在现有“无文件所有者字段”的模型下暂定仅管理员可用，下载保持公开。
- 存储服务校验空文件、图片 MIME 白名单和扩展名白名单，文件名由服务端生成。
- 存储、读取和删除都执行标准化路径与根目录包含检查，阻断 `..` 路径穿越。

## 剩余风险

- MIME 来自请求头，仍应增加文件魔数检测和病毒扫描；若恢复用户自行删除，必须先增加文件元数据和所有者字段。
